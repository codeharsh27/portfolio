The Problem

Campus placements are often a "blind" process. Students apply to hundreds of companies without knowing if their resume actually matches the job description. Furthermore, skill gaps are often discovered too late, leading to high rejection rates.

The Solution

DevApp bridges the gap using an offline-first mobile app. It features an AI Career Mentor for 24/7 guidance and a Skill Gap Analysis engine that scans user profiles against real market demands to recommend personalized learning paths.

Key Features

AI Skill Gap Analysis

Instantly identifies missing skills compared to target job descriptions.

Career Automation

Auto-Apply system that finds roles and tracks application status.

Offline First

Built with Hive to allow learning and tracking without internet.

AI Resume Builder

Generates ATS-optimized resumes and portfolio sites instantly.

AI Integration & Tech

The app follows Clean Architecture. The backend utilizes Firebase Cloud Functions and Local ML for intelligence. User data is secured with AES Encryption and stored locally in Hive boxes, ensuring fast access and privacy.

Tools Used

Flutter Firebase Hive Riverpod Clean Architecture
